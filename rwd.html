Responsive Portfolio Project: Class Assignment

Welcome to my final project for the Web Design for Everybody course! Throughout this journey, I've delved into HTML, CSS, and responsive web design principles, honing my skills to create engaging front-end experiences.

Project Overview

For our final assignment, we were tasked with crafting a three-page website from scratch, utilizing a template, or leveraging a framework. The website had to incorporate various layouts for different screen sizes and showcase at least four images on one of the pages. Additionally, we were encouraged to apply four "extra" front-end skills to our designs.

My Approach

I chose to build my website from the ground up, crafting three distinct pages: an about me page, a project page, and a resume page. Each page was meticulously designed to balance aesthetics with functionality, offering users a seamless browsing experience.

Innovative Features

To elevate my design, I implemented several advanced front-end techniques:

  1. Sticky navigation bar and footer: Enhancing navigation and accessibility across all pages.
  2. Pseudo element: Leveraging CSS wizardry to add depth and visual interest to elements.
  3. Prefix application: Ensuring cross-browser compatibility and smooth rendering.
  4. 3D image flip: Incorporating an eye-catching animation to captivate users.
